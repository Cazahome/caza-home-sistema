☁️ Integração Microsoft OneDrive

Conecte sua conta Microsoft para sincronizar todos os dados automaticamente no OneDrive. Os dados são salvos na pasta CazaHome/caza_home_dados.json no seu OneDrive.

✅ Configuração Azure já feita. URIs cadastradas: https://claude.ai e http://localhost
⚠️ Você está abrindo o arquivo localmente (file://). O login Microsoft não funciona assim.

✅ Solução recomendada — GitHub Pages (gratuito):
  1. Acesse github.com → crie uma conta grátis
  2. Crie um repositório chamado caza-home-sistema (público)
  3. Faça upload do arquivo renomeado para index.html
  4. Vá em Settings → Pages → Branch: main → Save
  5. Seu sistema ficará em: https://[usuário].github.io/caza-home-sistema/
  6. No Azure, adicione essa URL como URI de redirecionamento SPA
💡 Alternativa rápida: Abra o terminal na pasta do arquivo e execute:
python -m http.server 80 → acesse http://localhost/caza_home_sistema.html

Você precisará autorizar o acesso ao OneDrive para leitura e escrita de arquivos.
